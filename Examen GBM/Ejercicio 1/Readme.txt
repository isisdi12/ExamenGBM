Buenas tardes el siguiente documento es para explicarles las instrucciones
necesarias para correr el archivo Ejercicio1.py

1. Ingresar cualquier palabra que desee mientras esta tenga letras.

2. Solo se admite una palbra por prueba para confirmar si esta es palindroma o no.

3. Puede ingresar palabras tanto com mayúsculas o minúsculas pero debe ser completa por ejemplo: ANA ó ojo 
    NO ingrese con ambas ya que el programa lo tomará como una palabra que no es palindroma. Se adjunto un ejemplo
    llamado "Ejemplo Instrucciones Especificas" 

Espero que lo disfruten.
